# 🎬 YouTube Auto Pause (Chrome Extension)

A Chrome extension that automatically pauses YouTube videos when the tab is inactive and resumes playback when the tab becomes active. This helps improve productivity, reduce distractions, and save system resources.

---

## 🚀 Features

* ⏸️ Auto-pause YouTube videos when tab is inactive
* ▶️ Resume playback when tab becomes active
* ⚡ Lightweight and fast
* 🎯 Improves focus and reduces distractions
* 💻 Optimizes CPU and memory usage

---

## 🛠️ Tech Stack

* JavaScript
* HTML
* CSS
* Chrome Extension APIs

---

## 📦 Installation (Manual)

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/youtube-auto-pause.git
   ```

2. Open Chrome and go to:

   ```
   chrome://extensions/
   ```

3. Enable **Developer Mode** (top right)

4. Click on **Load unpacked**

5. Select the project folder

6. Done ✅ — Extension is now active!

## 🧠 How It Works

The extension uses Chrome's tab activity APIs to detect when a tab becomes inactive or active.
When a YouTube tab loses focus, the video is automatically paused.
When the tab becomes active again, playback resumes.
